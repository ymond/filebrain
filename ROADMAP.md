# Filebrain Roadmap

## Layer 1 — File Brain (current)
- [x] Project scaffolding, ollama setup
- [x] Extractor interface + PlainText, PDF, Code extractors
- [ ] SQLite metadata store
- [ ] ChromaDB vector store
- [ ] Text chunking + embedding pipeline
- [ ] File watcher daemon
- [ ] Query interface (semantic search + RAG with citations)
- [ ] CLI (`filebrain scan`, `watch`, `query`, `status`)
- [ ] Audio/video transcription extractor (stretch)

## Layer 2 — Life Logger
- [ ] Terminal session capture
- [ ] Browser extension for history/bookmarks
- [ ] Screen recording + storage
- [ ] Audio capture (Owl contribution / open-source Limitless replacement)
- [ ] All inputs feed into the same extraction → embedding pipeline

## Layer 3 — Active Assistant
- [ ] Chime-in capability (voice responses to overheard requests)
- [ ] Calendar integration
- [ ] Grocery list / task management from voice
- [ ] Zigbee smart home control
- [ ] Contextual reminders based on life awareness

## Layer 4 — Physical World

- [ ] Physics modeling engine
- [ ] Garden monitoring (cameras, observations, recommendations)
- [ ] Drone fleet integration
- [ ] Robotics attachment design
